# Executive Operations Digital Systems

Central access portal for the digital systems and institutional platforms of Partido State University’s Office of the Vice President for Executive Operations and Institutional Development.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy the `portal` directory to Vercel. No environment variables or backend services are required.

## Adding a System

Edit `src/data/systems.ts` and append a new entry to the `digitalSystems` array.
